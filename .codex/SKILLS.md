# ABEL OS+ Skill Catalog

Skills are loaded on-demand by the Specialist Agents.

## Core Multi-Environment Runners

- **Linux Primary**: Default execution environment via Docker/WSL.
- **Windows Worker**: Remote execution for Windows-specific UI/automation tasks.

## Tooling & MCPs

### Development & Navigation

- **Chisel**: High-efficiency diff editing (20-100x context savings).
- **Srclight**: Semantic codebase indexing and symbol search.
- **MCP Language Server**: Definition/Reference/Renaming via LSP.

### Web & Documentation

- **Playwright MCP**: Headless automation with isolated profiles.
- **Context7**: Live documentation and example injection from library URLs.
- **Brave Search MCP**: Privacy-focused real-time web knowledge.

### Persistent Memory

- **RecallNest Pattern**: Local-first hybrid memory using knowledge graphs (SQLite + FTS5).

### Android Security Lab

- **JADX Server**: Structured APK decompilation.
- **APKTool-MCP**: Resource modification and building.
- **Ghidra MCP**: Native binary analysis.
